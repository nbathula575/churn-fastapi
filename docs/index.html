Engineering Notes (how I think)

  • Offline validation: lock metrics before deployment (ROC-AUC, calibration, slice checks).
  • Thresholding: choose based on business cost, not habit.
  • Production monitoring: track data drift + prediction drift + latency + error rate.
  • Reproducibility: version artifacts + pin dependencies + containerize.
  • Failure modes: bad inputs, schema drift, cold start latency, silent data leakage.
